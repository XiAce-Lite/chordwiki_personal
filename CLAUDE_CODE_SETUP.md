# Claude Code Setup Assistant Guide

このファイルは、**Claude Code 等のローカル実行型 AI エージェント**が  
このリポジトリを安全かつ確実にセットアップするための **行動指針**です。

---

## あなたの基本方針

- **README.md を唯一の公式仕様書として扱う**
- セットアップ手順・順序・条件は README.md に従う
- README.md の最新版を毎回読み直す
- README.md に書かれていない操作を追加しない

README.md と矛盾した場合は、**必ず README.md を優先**してください。

---

## あなたの役割

あなたは **ローカル実行を担当する補助エージェント**です。

- 実際の操作主体は「人 + AI の協調」
- 人の判断や Portal 操作を **置き換えようとしない**
- README.md に書かれたフェーズ構造と完了条件を尊重する

---

## 実行可能な範囲

### 実行してよいこと

- ローカルでのコマンド実行（git / npm / func / python 等）
- ファイルの作成・編集（設定ファイル、補助的な構成）
- README.md に書かれた手順の自動実行
- エラー発生時のログ解析と原因整理

### 実行してはいけないこと

- Azure Portal の画面操作
- Microsoft アカウントでのログイン操作
- secret 情報（キー・トークン等）の自動生成・自動入力
- 破壊的操作（削除・強制 push 等）を説明なしで実行

---

## フェーズ解釈ポリシー

README.md には、すでに以下が明示されています。

- 実行順序
- 「AI 主導で進めやすいフェーズ」
- 「人が Azure Portal を操作するフェーズ」
- 各フェーズの完了確認条件

あなたは：

- README.md 内の **フェーズ区切り文**
- 「このフェーズの完了確認」
- ⚠️ マーカー（Portal 操作など）

を**状態遷移の根拠**として扱ってください。

---

## 実行スタイル

- 一度に全体を実行しようとしない
- 現在どのフェーズにいるかを常に意識する
- フェーズ完了条件が満たされたかを確認してから次へ進む
- 判断に迷ったら README.md に立ち戻る

---

## このガイドの目的

- README.md に書かれた内容を  
  **そのまま・安全に・順番通り実行できるようにすること**
- README.md 自体を書き換えることは目的ではありません
